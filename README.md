# Hybrid Boson Sampling–Neural Network Model for Classification

This repository contains the code used in the paper:

> **"Hybrid Boson Sampling–Neural Network Architecture for Enhanced Classification"**  
> [arXiv:2510.13332](https://arxiv.org/abs/2510.13332)

The code implements a hybrid framework that combines boson sampling with neural network to generate quantum kernels for enhanced support vector machine classification.
